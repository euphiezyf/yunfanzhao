From the observation of using competitive apps and LinkedIn message components, we found several pain points and redesign ideas below.

Voice Message:

  • Adding feedback on whether listened or not.

    • Text Message:

  • Auto-reply function is not as helpful as it posits.
  • Showing the status of read and unread.
  • Adding reply to a message function.

    • Other Message:

  • Adding contacts card sharing, which might be good for referral.

    • Features:

  • Adding search chat history, view media and files, archive, block, delete, and report.

    • Design:

  • Using chat bubbles for clear text separation.
  • Eliminating redundant names.
